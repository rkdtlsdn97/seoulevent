# seoulevent

## 프로젝트 설명
SeSAC(청년취업사관학교)에서 진행 된 프로젝트로 서울 공공데이터 api를 활용하여 서울의 문화행사를 쉽게 찾아주는 프로젝트

<br/>

###배포 링크 https://seoulevent.vercel.app/

## 구현기능
서울의 지역과 장르를 선택하면 선택된 항목에 맞는 문화행사들 나열

<br/>

## 사용 skill
HTML, CSS, JavaScript
